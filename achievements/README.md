# Achievements

Monthly summaries of what I've been working on across GitHub. I use these for performance reviews, reflections, and keeping track of where my time actually goes.

## How it works

A cron job runs on the 1st of each month, pulls my GitHub activity (PRs, issues, reviews, comments) from a set of repos, and dumps the raw data into a [private repo](https://github.com/lewis-mcgillion/career-data). Then a Copilot coding agent reads that data and writes up a summary here — stripped of any internal details so it's safe to keep public.

See [SETUP.md](../SETUP.md) if you want to set this up for yourself.
